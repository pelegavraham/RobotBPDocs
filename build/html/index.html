Welcome to RobotBP’s documentation!¶

Here you will learn how to actuate robot based on raspberry-pi supported EV3 Lego and Grove-pi kit with behavioral programming written in Javascript, whats called BPjs.

If you are new to BPjs you can learn from the tutorial in: BPjs tutorial

In additional, you will learn on several ways how to extend the existing project.
