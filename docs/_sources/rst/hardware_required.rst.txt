.. _parts:

Hardware required
=============================================

As mentioned, RobotBP help you to write code that acturate your robot with BPjs.

The parts the robot consists of them are:

Rasspbery Pi
---------------------------------------------


The Raspberry Pi is a series of small single-board computers has evolved through several versions that feature variations 
in the type of the central processing unit, amount of memory capacity, networking support, and peripheral-device support.

For more information click `here <https://en.wikipedia.org/wiki/Raspberry_Pi>`__.

.. figure:: /images/Raspberry.jpg

Lego Mindstorms EV3
---------------------------------------------

Lego Mindstorms EV3 (evolution 3) is the third generation robotics kit in Lego's Mindstorms line.
The EV3 Home (31313) set consists of: 1 EV3 programmable brick, 2 Large Motors, 1 Medium Motor, 1 Touch Sensor, 1 Color Sensor, 1 Infrared Sensor, 1 Remote Control, cables, USB cable, and 585 TECHNIC elements.

For more information click `here <https://en.wikipedia.org/wiki/Lego_Mindstorms_EV3>`__.

.. figure:: /images/EV3.jpg

Grove Pi
---------------------------------------------
GrovePi is an add-on board that brings Grove Sensors to the Raspberry Pi.
It is consists of: 7 digital Ports, 3 analoge Ports, 3 I2C ports, 1 Serial port connect to GrovePi, 1 Serial port connect to Raspberry Pi and Grove header Vcc output Voltage: 5Vdc

For more information click `here <https://wiki.seeedstudio.com/GrovePi_Plus/>`__.

.. figure:: /images/grove.jpg

.. admonition:: Note!

   The Raspberry Pi is MUST to be installed as part of the robot. 

   The EV3 and Grove Pi are additional recommended parts, to get more exciting actions from your robot.
