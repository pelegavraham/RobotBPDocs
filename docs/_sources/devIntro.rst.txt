Introduction
==============

Our project support in some devices with some functionality, depending on time constraints.

We have done our best to allow you to expand the existing project in the most convenient and efficient way.

In the following tutorial we present to you the simplest way to expand the project by:

* adding new command in the system

* adding new board in the system

* adding new queue to transmit additional messages type in the system

Finally, we will show you some important points you should know.

**Enjoy!**
