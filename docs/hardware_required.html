Hardware required¶

As mentioned, RobotBP help you to write code that actuate your robot with BPjs.

The parts the robot consists of are:

Raspberry Pi 3¶

The Raspberry Pi is a series of small single-board computers has evolved through several versions that feature variations in the type of the central processing unit, amount of memory capacity, networking support, and peripheral-device support.

For more information click here.

Note!

The Raspberry Pi require MicroSD 32 GB for the installation of the Raspberry Pi Imager. More tools that you need for the installation can be found here.

Lego Mindstorms EV3¶

Lego Mindstorms EV3 (evolution 3) is the third generation robotics kit in Lego’s Mindstorms line. The EV3 Home (31313) set consists of: 1 EV3 programmable brick, 2 Large Motors, 1 Medium Motor, 1 Touch Sensor, 1 Color Sensor, 1 Infrared Sensor, 1 Remote Control, cables, USB cable, and 585 TECHNIC elements.

For more information click here.

Grove Pi¶

GrovePi is an add-on board that brings Grove Sensors to the Raspberry Pi. It is consists of: 7 digital Ports, 3 analoge Ports, 3 I2C ports, 1 Serial port connect to GrovePi, 1 Serial port connect to Raspberry Pi and Grove header Vcc output Voltage: 5Vdc

For more information click here.
